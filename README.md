This repository contains the code and calibration coefficients to correct seal-borne CTD tags from the PS146 Polarstern expedition to the Weddell Sea.
We deployed two types of CTD tags on Weddell seals (<i>Leptonychotes weddellii</i>): 
1. Wildlife Computers SCOUT CTD tags
2. Sea Mammal Research Unit (SMRU) CTD tags.

We placed the CTD tags on the ship's CTD rosette during routine ship casts to calculate calibration coefficients to correct pressure, conductivity and temperature for
each tag. Corrections and calibration coefficients are unique to each tag.
Methods for callibration, formulas for the corrections and more detail can be found in Chapter 4, pages 107-127 of the PS146 expedition report here: https://epic.awi.de/id/eprint/60421/
